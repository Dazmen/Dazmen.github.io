Sweet Eats Bakery!

This was the very first project I worked on as a Lambda Student. It was even done on codepen!

The goal was to develop and show an understanding of semantic HTML and the use of fundamental CSS to style. We also had to learn how to use Git in order to submit the project through Github. At the end of this project we were expected to know how to

  • describe and use semantic HTML tags to markup content
  • demonstrate the use of CSS selectors to style markup
  • demonstrate CSS specificity and inheritance to create and update styles
  • explain why we use the command line interface (CLI), and demonstrate use of basic CLI commands.
  • sufficiently explain what version control and Git are, and why as software engineers we use them.
  • understand branching, and be able to create a branch locally.
  • explain and properly utilize the git commands: `status`, `add`, `commit`, and `push`

As a new student who has very little prior coding experience, this was a lot to take in! Lambda School likes to keep the pressure on, no slacking!
