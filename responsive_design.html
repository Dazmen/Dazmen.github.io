Responsive Design! @media

This project was for the aspect of responsive design!.

The goal was to develop and show an understanding of responsive web design and media queries to format a web page to look great on any device. At the end of this project we were expected to know how to

  • describe the differences in fixed, fluid, adaptive, and responsive layouts.
  • demonstrate knowledge of media queries by correctly implementing them in a project.
  • build responsive layouts using an effective approach.
  • describe and use responsive units to create a fully responsive website
  • describe why responsive design impacts accessibility

Building a responsive and accessible website may take more work, but it is worth it! It is becoming more important every year to build websites that are accessible to everyone, regardless of device or impairment!
